# VroomVroom

Jeu de rapidité avec balle et obstacle codé en python.
Ce programme a été réalisé avec Python et Processing ainsi que d'autres logiciels pour la décoration.
C'est mon premier programme python réalisé principalement par moi même et de l'accompagnement d'un camarade.

Télécharger dans releases le fichier .rar (ouvrir avec winrar) et lancer avec l'éxécutable pour le lancement du jeu.
Le jeu est sous 2 versions, WASD et ZQSD pour les principales directions.

Le but du jeu est donc d'arriver le plus rapidement à la case de fin.
Les collisions avec les obstacles, murs pénalise le joueur avec ajout de temps à son score.

Le record en temps est 4.1s donc bonne chance !
